# VS Code 中的 Git

当编辑器打开项目文件的更根目录时，如果包含有 `.git` 文件，则会自动识别并标上符号与颜色。

- D 表示已删除的文件 Deleted
- U 表示未跟踪的文件 Untracked
- M 表示修改过的文件 Modified

其他：

- R 表示更名了的文件 Renamed（暂存了更名前和更名后的文件改动后会显示此标记）
- A 新建的文件 Added（从零开始的文件）
